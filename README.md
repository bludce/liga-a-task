# liga-a-task

### Задача
[Написать одностраничное приложение с использованием React+hooks, выводящее список постов](https://www.notion.so/000f70ef4b5a4327a8621c7ab8bbeec2)

### Технологии
* React (hooks)
* Typescript
* Styled-components

### Установка
Установите зависимости и запустите сервер.
```sh
$ cd liga-a-task
$ npm install
$ npm start
```
